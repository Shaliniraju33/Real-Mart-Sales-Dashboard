# Real-Mart-Sales-Dashboard

## 📌 Project Overview

The Real Mart Sales Dashboard is an interactive Power BI project designed to analyze retail sales data and provide valuable business insights. It helps in understanding sales performance, customer trends, and product analysis through visually rich and interactive dashboards.

---

## 🎯 Objective

* Monitor overall sales performance
* Analyze product and category-wise sales
* Identify high-performing regions
* Support data-driven business decisions

---

## 📊 Dataset

The dataset includes retail sales data such as:

* Product Name
* Category
* Sales Amount
* Quantity Sold
* Profit
* Customer Location
* Order Date

---

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel / CSV
* Power Query (Data Cleaning & Transformation)

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Transformation using Power Query
4. Data Modeling
5. Dashboard Design
6. Visualization & Insight Generation

---

## 📈 Dashboard Features

* KPI Cards (Total Sales, Total Profit, Total Orders)
* Sales Trend Analysis (Monthly / Yearly)
* Category-wise Performance
* Region-wise Analysis
* Interactive Filters (Slicers)
* Drill-down Functionality

---

## 🔍 Key Insights

* Identified top-selling products and categories
* Observed seasonal sales trends
* Analyzed regional performance
* Highlighted profit and loss areas

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use filters and visuals to explore insights
