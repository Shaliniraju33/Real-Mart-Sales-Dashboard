# Real-Mart-Sales-Dashboard
Overview

This project is an interactive Real Mart Sales Dashboard built using Power BI to analyze and visualize sales performance. It provides a clear understanding of key business metrics such as total sales, profit, and order details.

Features
Sales and Profit Analysis
Product and Category Performance
Region-wise Insights
Monthly and Yearly Trends
Interactive Filters and Slicers
Tools Used
Power BI
Power Query
Objective

The objective of this project is to present sales data in a simple and interactive way, helping users understand performance, identify trends, and make better business decisions.
